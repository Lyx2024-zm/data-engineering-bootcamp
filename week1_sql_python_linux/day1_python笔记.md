## 1. 数据类型
- 整数（int）：如 1, 100, -5
- 浮点数（float）：如 3.14, -0.5
- 字符串（str）：文本，必须用引号包裹，如 "hello"
- 布尔值（bool）：True 或 False

## 2. 基本操作
```python
a = 10

c = "数据工程"
d = True
```
